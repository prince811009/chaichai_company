Theming

Assan included 8+ theme styles with different colors, fonts but have same components you viewing in preview

How to use theme styles?
  1. 01 - Create a new .html page into your project
  2. 02 - Visit any of these theme styles from demo located in navbar top - hover on features/
  3. 03 - Scroll down a bit on this page, click on Copy Code button, paste into the page you just created into your project
  4. 04 - That's it! You just created a getting started landing page for your project using Assan theme with required css/js files included. Congrats!!
How to update colors and fonts?
  1. 01. Well! If you're familiar with the modern day tools like Sass for Css, It's going to be a very easy for you to update the look and feels of your landing page.
  2. 02 - Open theme style related _variables.scss file from scss/custom/variables/
    For example - You've choose the theme style purple layout for your project, You need to open scss/custom/variables/_variables-theme-purple.scss file
  3. 03 - First we check which color name we use in project's primary color (purple theme style example continues..)
  4. 04 - After that we search primary color name value (its appears in top after grays)
    If your project brand color anywhere near like purple color - Change purple color value and hit save. Thats it!
    You can also rename your theme style name and use any color value for your project's brand color. above are just an idea of changing colors.
  5. Updating fonts
    Find bellow variable values in _variables.scss file for updating the fonts
    Update these values with your brand font family.
    $font-family-serif is the secondary font option uses for blog articles, testimonials etc...
